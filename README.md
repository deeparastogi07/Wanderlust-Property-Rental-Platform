# Wanderlust – Property Rental Platform

A full-stack web application that allows users to explore, list, and review rental properties.
The platform provides secure authentication, property listing management, and a review system similar to modern rental marketplaces.

---

## Features

* User authentication and authorization using Passport.js
* Create, edit, and delete property listings
* Review and rating system for properties
* Image upload and storage using Cloudinary
* Responsive UI built with Bootstrap
* RESTful APIs with MVC architecture
* Secure session management

## Installation

1. Clone the repository

```
git clone https://github.com/deeparastogi07/Wanderlust-Property-Rental-Platform.git
```

2. Navigate to the project folder

```
cd Wanderlust-Property-Rental-Platform
```

3. Install dependencies

```
npm install
```

4. Run the server

```
node app.js
```

5. Open in browser

```
http://localhost:3000
```
